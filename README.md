# Diablo Helper

간단하고 직관적인 **Diablo 매크로 헬퍼 프로그램**입니다.  
Python 기반 GUI로 제작되었으며, 키보드와 마우스 입력을 **순차적·반복적으로 자동 실행**할 수 있습니다.

> ⚠️ 본 프로그램은 개인 학습 및 테스트 용도로 제작되었습니다.  
> 게임 내 매크로 사용은 이용 약관 위반 및 제재 대상이 될 수 있습니다.

---

## ✨ 주요 기능

- 🔁 키 입력 순차 반복  
  `1 → 2 → 3 → 4 → 마우스 좌클릭 → 마우스 우클릭`

- 🎚 개별 입력 딜레이 조절  
  각 키/마우스 입력마다 **개별 속도 설정 가능 (초 단위)**

- 🔄 사이클 딜레이 설정  
  한 사이클 종료 후 대기 시간 설정 가능

- 🔘 ON / OFF 토글 방식  
  `Pause` 키로 매크로 시작 / 중지

- 🖥 GUI 기반 조작  
  체크박스로 원하는 입력만 선택 가능

- 📌 상태 표시  
  현재 실행 상태를 GUI에서 실시간 확인

---

## 🧰 사용 기술

- **Python 3**
- **Tkinter** – GUI
- **pyautogui** – 키보드 / 마우스 자동 입력
- **keyboard** – 전역 키 입력 감지
- **threading** – 멀티스레드 처리
